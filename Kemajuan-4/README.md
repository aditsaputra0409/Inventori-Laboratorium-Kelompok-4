# PROGRES 4: IMPLEMENTASI DAN LAPORAN AKHIR

**Mata Kuliah:** Sistem Basis Data
**Studi Kasus:** Sistem Manajemen Inventori Laboratorium
**Kelompok:** 4

## Deskripsi

Progres 4 merupakan tahap akhir pengembangan Sistem Manajemen Inventori Laboratorium. Pada tahap ini dilakukan implementasi basis data secara lengkap menggunakan MySQL/MariaDB, pengujian seluruh fitur database, serta penyusunan laporan akhir proyek. Seluruh komponen database telah berhasil dibuat mulai dari perancangan ERD, normalisasi, pembuatan tabel, relasi antar tabel, pengisian data, hingga pengujian query SQL.

## Capaian Proyek

Pada tahap ini kelompok berhasil menyelesaikan:

* Merancang database Sistem Manajemen Inventori Laboratorium.
* Membuat Entity Relationship Diagram (ERD).
* Melakukan normalisasi data hingga Third Normal Form (3NF).
* Mengimplementasikan tabel menggunakan SQL DDL.
* Mengimplementasikan data menggunakan SQL DML.
* Membuat relasi antar tabel menggunakan Primary Key dan Foreign Key.
* Melakukan pengujian query SELECT.
* Melakukan pengujian query JOIN.
* Melakukan pengujian query GROUP BY.
* Melakukan pengujian query UPDATE dan DELETE.
* Menyusun laporan akhir proyek.
